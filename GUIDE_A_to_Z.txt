  PART 4 — LOAD CELL CALIBRATION (Pehli baar karna hai)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. ESP32 code mein ye line change karo:
         #define CALIBRATION_MODE   true

  2. Upload karo → Serial Monitor kholo (115200 baud)

  3. Koi weight nahi rakhna — Serial pe ek number aayega
     Note karo: zero_value = ___________

  4. Apna 1kg ya known weight rakho
     Note karo: with_value = ___________

  5. Formula:
     CALIBRATION_FACTOR = (with_value - zero_value) / weight_in_kg

     Example: (230000 - 1500) / 1.0 = 228500

  6. Wo number code mein daalo:
         #define CALIBRATION_FACTOR   228500.0f

  7. CALIBRATION_MODE wapas false karo:
         #define CALIBRATION_MODE   false

  8. Dobara upload karo — ab accurate readings aayengi!

  PART 10 — TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problem: OLED kuch nahi dikh raha
  ─────────────────────────────────
  → SDA = GPIO 21, SCL = GPIO 22 check karo
  → VCC = 3.3V (5V nahi!)
  → I2C address 0x3C ya 0x3D try karo

  Problem: HX711 readings 0 ya bahut galat
  ─────────────────────────────────────────
  → CALIBRATION_MODE = true karke calibrate karo
  → Load cell wiring: Red=E+, Black=E-, White=A+, Green=A-
  → DT = GPIO 16, SCK = GPIO 17

  Problem: WiFi connect nahi ho raha
  ───────────────────────────────────
  → WIFI_SSID = "HUAWEI-yJ9e" (capital letters check karo)
  → WIFI_PASSWORD = "hammad123" (sahi hai?)
  → ESP32 2.4GHz WiFi use karta hai — 5GHz nahi chalega
  → Router se zyada door na ho

  Problem: Firebase error
  ────────────────────────
  → FIREBASE_HOST mein https:// nahi hona chahiye
  → Rules check karo: .read = true, .write = true
  → Firebase Console mein Realtime Database enabled hai?

  Problem: Website pe "Device not found"
  ──────────────────────────────────────
  → ESP32 ON hai aur WiFi connected hai?
  → Device ID bilkul sahi daala? (Capital letters, no spaces)
